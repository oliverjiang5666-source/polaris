### 合成数据的参数来源
- 山东2024年日前出清均价: **316.21 元/MWh**（官方）
- 山东2024年实时出清均价: **309.52 元/MWh**（官方）
- 山东2023年发电侧结算均价: **468.16 元/MWh**（官方）
- 广东基准电价: ~350 元/MWh（行业估算）
- 峰谷价差、负电价频率等: 基于行业报告统计

### ⚠️ 合成数据的局限
1. **日内形状是估算的** — 真实价格曲线更不规则，跟天气/负荷强相关
2. **缺少极端事件的真实分布** — 负电价持续时长、尖峰价格的自相关性
3. **没有辅助服务市场数据** — 调频里程价格、调峰补偿价格
4. **没有中长期合约价格** — 储能参与的中长期曲线分解

## 获取真实数据的3条路径

### 路径一（推荐）: 电查查 API 💰
- **网站**: https://dianchacha.cn
- **联系**: 18035768255 / dianchacha@mlogcn.com
- **数据**: 15分钟/1小时粒度的现货出清价格，覆盖全省
- **价格**: 预估 2000-5000 元/月
- **优势**: 数据最全、最规范、有API
- **行动**: 先注册免费账号看看能白嫖什么

### 路径二: 交易中心 Selenium 抓包 🔧
- **目标**: 山东 pmos.sd.sgcc.com.cn / 广东 pm.gd.csg.cn
- **前提**: 需要市场主体账号（企业资质）
- **方法**: 
  1. `pip3 install selenium selenium-wire`
  2. `python3 scrape_real_data.py capture`
  3. 手动登录，脚本自动抓取API endpoint
  4. 拿到endpoint后写批量拉取脚本
- **优势**: 免费、数据最原始
- **劣势**: 需要企业账号，API可能随时变
- **行动**: 找一个有账号的合作方帮忙抓一次endpoint

### 路径三: 月度报告手工整理 📊
- **来源**: 北极星(bjx.com.cn)、能源局(nea.gov.cn)、CNESA
- **数据**: 月度粒度的均值数据
- **优势**: 完全公开、无需账号
- **劣势**: 粒度太粗（月级），无法做15min级backtest
- **适用**: 验证合成数据的统计参数是否靠谱

## 数据字段标准（对齐ERCOT格式）

```python
# ERCOT数据格式
{
    "timestamp": "2024-01-15T08:15:00-06:00",  # 15min
    "settlement_point": "HB_WEST",
    "spp": 45.23,                               # $/MWh
}

# 中国数据格式（目标）
{
    "timestamp": "2024-01-15T08:00:00+08:00",   # 1h (日前) 或 15min (实时)
    "province": "shandong",
    "node": "SD_UNIFIED",
    "da_price": 316.21,                          # 元/MWh (日前)
    "rt_price": 309.52,                          # 元/MWh (实时)
    "market_type": "spot",
}
```
